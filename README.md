# Template

Template creation for Web Development Project.

Using [node.js](https://nodejs.org/en/).

- Clone Project,
- Installation : npm i (cmd),
- gulp (cmd) to launch browserSync and run all services.

## Dependencies npm

- Gulp
- Sass
- Gulp-rename
- Sourcemaps
- Autoprefixer
- Uglify
- BrowserSync
